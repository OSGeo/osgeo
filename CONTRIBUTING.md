# Contributing to OSGeo
Thank you for your interest in contributing to the Open Source Geospatial Foundation (OSGeo).


## How to Contribute
1. Fork this repository to your GitHub account.
2. Clone your fork locally.
3. Create a new branch for your changes.
4. Make your changes with clear and meaningful commits.
5. Push your branch to GitHub.
6. Open a Pull Request describing your changes.


## Contribution Guidelines
1) Keep changes small and focused.
2) Follow existing formatting and structure.
3)  Ensure content is clear and professional.
4) Be respectful and follow OSGeo community guidelines.
